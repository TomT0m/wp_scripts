Set of pywikipedia pwbscripts

Wikipedia (french Wikipedia):
* cleaning Wikiprojects Homepage discussions from deletion and fusion proposition of articles
* maintain the state of a list of open requests in the Wikiproject Announce Page

Wikidata :
* various one (or more) time pwbscripts to import sequences of page like sequences of series seasons or govenments as sequence of statements followed by/preceeded by on Wikidata.
* more

* mwparserfromhell
* python yaml parser (python-yaml package on debian)
* dirspec : library to find configuration filepath in different OSes
 * for windows : pywin32
* httplib2
